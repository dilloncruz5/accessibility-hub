## About The Project

**Accessibility Hub** is a modern website offering a comprehensive suite of digital accessibility tools. It provides Text-to-Speech, Speech-to-Text, Braille support, keyboard navigation, and various vision aids like a screen reader mode and high-contrast themes, all designed to make the web a more inclusive place.

---

## ✨ Key Features

-   **🗣️ Text-to-Speech**: Convert text into natural-sounding speech.
-   **🎤 Speech-to-Text**: Real-time voice transcription with high accuracy.
-   **⠗ Braille Support**: On-screen text-to-braille conversion.
-   **⌨️ Full Keyboard Navigation**: Navigate the entire interface without a mouse.
-   **👁️ Vision Aids**: Includes high-contrast themes, color adjustments, and font scaling.
-   **🎨 Light & Dark Modes**: Switch between themes for visual comfort.

This project is built with a modern and robust technology stack:

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
